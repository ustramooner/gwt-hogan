Examples of loading data:

Test 2, field 1(without html formatting): {{data2.field1}}
Test 1, field 2(using html formatting): {{{data2.field1}}}

A JSONArray list:
    {{#list}}
  • {{.}}
    • {{/list}}



And of course you can do everything else you can using Hogan (or more specifically, mustache, since that's the engine that Hogan uses). For more info see https://github.com/janl/mustache.js
